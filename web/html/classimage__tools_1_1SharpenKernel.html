Detailed Description

This class implements an sharpen kernel for use with a ConvolutionFilter. init_kernel() constructs the sharpen kernel for this class. The middle kernel coord will be set to the square of the kernel's dim. This is one greater than the sum of all other coord values. All other coords set to -1. See example:

-2 -2 -2 -2 17 -2 -2 -2 -2
